## Reviewing Generated Content

After generation completes:

1. **Review the Draft**
   - Click "Edit in WordPress"
   - Check for accuracy
   - Verify tone and style
   - Confirm citations

2. **Make Edits**
   - Edit directly in WordPress
   - Adjust formatting
   - Add/remove content
   - Update images

3. **Publish**
   - Use WordPress preview
   - Schedule or publish
   - Share with team

## Best Practices

### Writing Effective Prompts

**Good:**
- "Virginia's new election integrity measures and their impact on Goochland"
- "Recent school board meeting decisions on curriculum"

**Avoid:**
- "Politics"
- "Stuff from the meeting"

### Review Checklist

Before publishing, verify:
- [ ] Facts are accurate
- [ ] Sources are cited
- [ ] Local connection is clear
- [ ] Tone is appropriate
- [ ] No typos or errors
- [ ] Image is relevant
- [ ] Call-to-action is included

### Quality Tips

1. **Provide Context:** More context = better results
2. **Be Specific:** Specific topics generate better content
3. **Review Carefully:** Always review before publishing
4. **Edit Freely:** Generated content is a starting point
5. **Save Sources:** Keep original documents referenced

## Troubleshooting

### Generation Failed
- Check WordPress connection
- Verify file format (PDF, DOCX, TXT only)
- Try smaller file size
- Refresh page and retry

### WordPress Connection Lost
- Verify credentials in settings
- Check WordPress site is accessible
- Contact administrator

### Content Quality Issues
- Provide more specific topic/context
- Include priority sources
- Add more detail to meeting minutes
- Regenerate with adjusted inputs
